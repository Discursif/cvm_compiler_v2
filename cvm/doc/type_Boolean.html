type Boolean

parent Bytes
size 1

Documentation

Boolean object accept only two values 0 and 1
Other values are possible but will cause UB

References

false
This reference is equivalent to 0
true
This reference is equivalent to 1

Methods

double_or (Boolean other) → Boolean
This is the implementation of || operator and will only return true if the current object or the input are set to true
to_string () → String
If true will return "true" and if false will return "false"
not () → Boolean
This is the implementation of ! operator if true will return false and reciprocally
double_and (Boolean other) → Boolean
This is the implementation of && operator and will only return true if the current object and the input are set to true
len () → Byte
Boolean length will always be equals to 1
merge (Bytes other) → Panic
⚠ This function if called will crash the compiler

You can't merge a Boolean with anything
